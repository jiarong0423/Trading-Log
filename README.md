# 📊 交易日誌系統

個人交易記錄與損益追蹤系統，支援股票、期貨交易記錄。

## 🌐 線上使用

訪問：[https://嘉架0423.github.io/交易日誌/](https://嘉架0423.github.io/交易日誌/)

## ✨ 功能特色

- 📈 支援個股、期貨交易記錄
- 💰 自動計算已實現/未實現損益
- 📊 日/週/月結算報表
- 💾 資料儲存在瀏覽器本地
- 📥 支援匯出 CSV 和 HTML

## 🔒 隱私說明

所有資料僅儲存在您的瀏覽器中（localStorage），不會上傳到任何伺服器。

## ⚠️ 重要提醒

請定期使用「下載日誌」功能備份您的資料！
